# Contributing

Thank you for your interest in this project.

## Scope
This repository is a small reproducible example for comparing lexicon-based sentiment methods in R. Contributions should keep the project simple, transparent, and easy to reproduce.

## Ways to contribute
- Report bugs or errors in the code
- Suggest improvements to documentation
- Propose clearer examples or visualizations
- Improve reproducibility and code clarity

## Guidelines
- Keep contributions focused on public-safe sample data and reproducible code
- Do not add private, restricted, or sensitive datasets
- Keep file structure simple and consistent with the existing repository
- Write clear commit messages

## Submitting changes
1. Fork the repository
2. Create a new branch for your change
3. Make your edits
4. Submit a pull request with a short explanation

## Code style
- Use clear and readable R code
- Prefer simple tidyverse-style workflows
- Keep documentation concise

## Questions
For major changes, open an issue first to discuss the proposed update.
