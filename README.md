## Overview

Reproducible R workflow comparing **AFINN**, **Bing**, and **NRC** lexicon-based sentiment analysis on social media text. Helps researchers understand how lexicon choice affects sentiment scores and classifications.

## Lexicons Compared

| Lexicon | Type | Output |
|---|---|---|
| **AFINN** | Numeric | -5 to +5 per word |
| **Bing** | Binary | Positive / Negative |
| **NRC** | Emotion | 10 emotions + polarity |
